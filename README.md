# Golem Gateway

Модульный API-маршрутизатор без сохранения состояния (stateless) для больших языковых моделей. Golem Gateway работает как невидимый мост между веб-интерфейсами ИИ и стандартными API-клиентами, предоставляя REST-интерфейс, совместимый с OpenAI, через автоматизацию headless-браузера (Puppeteer) и перехват XHR-потоков.

## Архитектура и возможности

- **Совместимость с OpenAI API:** Нативная поддержка эндпоинтов `/v1/models` и `/v1/chat/completions` (включая параметр `stream: true`). Работает как прямая замена для любого клиента, поддерживающего спецификацию OpenAI.
- **Модульная система провайдеров:** Базируется на архитектурном паттерне Router. Логика ядра строго отделена от реализаций конкретных нейросетей (папка `providers/`). Легко расширяется для добавления ChatGPT, Claude, Qwen и других.
- **Движок интеграции DeepSeek:**
 - Динамическое переключение моделей и модификаторов (`expert`, `search`, `think`) через взаимодействие с DOM.
 - Сетевой перехват событий Server-Sent Events (SSE) на низком уровне для бесперебойного потокового вывода токенов.
 - Автоматическая "стерильная" очистка: теневая сессия на целевой платформе удаляется сразу после завершения генерации ответа, предотвращая засорение истории аккаунта.
- **Изоляция контекста:** Результаты веб-поиска капсулируются внутри специфичных HTML-тегов (`<ds-search-results>`), что предотвращает загрязнение контекста в RP-клиентах.
- **Система автоматической авторизации:** Встроенный локальный модуль для безопасного, бесшовного внедрения свежих токенов сессий и cookie-файлов напрямую в конфигурацию `.env`.

## Требования

- [Node.js](https://nodejs.org/) (рекомендуется версия 16.0.0 или выше)
- Установленный Google Chrome или Chromium на хост-машине.

## Установка и запуск

### 🚀 Быстрый старт (Windows - Автоматически)
Для пользователей Windows предусмотрен запуск в один клик. Вам **не нужно** вручную устанавливать библиотеки или вводить команды в консоль:
1. Склонируйте репозиторий или скачайте исходный код в zip-архиве и распакуйте его.
2. Дважды кликните по файлу **`start.bat`**.
> Скрипт полностью автономен: он сам проверит наличие необходимых библиотек, автоматически скачает их через `npm install` и сразу запустит ядро маршрутизатора.

### Linux / macOS (Вручную)
1. Склонируйте репозиторий в нужную директорию.
2. Откройте терминал и установите зависимости:
 ```bash
 npm install
 ```
3. Запустите сервер:
 ```bash
 node start.js
 ```

*Примечание: При первом запуске (если действительные токены сессии отсутствуют в `.env`), система автоматически откроет локальную страницу настройки в вашем браузере для быстрой авторизации.*

## Интеграция с программным обеспечением (SillyTavern)

Настройте ваше подключение как `OpenAI Custom Endpoint`, указав адрес: `http://127.0.0.1:7777/v1`.

**Управление чистотой контекста:**
При использовании моделей с активным веб-поиском (например, `deepseek-fast-search`), маршрутизатор добавляет компактную сводку прочитанных страниц в конец ответа.
